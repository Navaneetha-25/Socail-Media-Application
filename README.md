# Social Media Application (Console Based)

## Project Description

This is a console-based Social Media Application developed in Java.
The system allows users to sign up, connect with friends, create posts, interact with posts, and manage notifications.
The application simulates basic social networking features using object-oriented programming concepts.

## Modules Implemented

1. User Management
2. Friend Suggestions
3. Friend Request Management
4. Post Management
5. News Feed
6. Notification Management
   
##  Functional Features

### User Signup

Users register by entering:
* Name
* Date of Birth
* Age
* Location
* Occupation

### Friend Suggestions

Users are categorized into:
* Mutual Friends (with mutual friend count and names)
* Matching Profiles (Name, Age, DOB month, Location – sorted by highest match)
* Other Users

### Friend Request Management

* Send friend requests
* View pending requests
* Accept / Reject requests
* Update friend lists upon acceptance

### View Friends

Displays:
* Name
* Age
* Location
* Occupation

### Post Management

* Create posts visible to friends
* Example: "Enjoying the sunny day in New York! #blessed"

### News Feed

* Shows posts from friends
* Displays:
  * Post content
  * Author
  * Like count
  * Comments
* Latest posts appear first

### Post Interaction

* Like posts
* Comment on posts
* Updated like count and comments shown instantly

### Share Post

* Share posts with other users
* Shared posts generate notifications
* Recipients can like, comment, or re-share

### Notification Management

Notifications generated for:
* Friend requests
* Accepted requests
* Shared posts
* Post interactions
Notifications are cleared once viewed or acted upon.

## Technologies Used

* Java
* OOP Concepts
* Collections Framework
* Console-based Interface

## How to Run

1. Compile the project.
2. Run the Main class.
3. Follow on-screen menu options.
